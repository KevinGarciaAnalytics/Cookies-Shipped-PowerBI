# Cookies-Shipped-PowerBI
A dashboard created using PowerBI to find information on cookies shipped.
## Project Overview
This project showcases the development of an interactive Business Intelligence dashboard using Power BI to analyze cookie sales performance and business metrics. The dashboard provides visual reporting through KPIs, charts, and filters, enabling users to monitor sales trends and overall business performance.

## Project Objective
Develop an interactive dashboard that allows users to monitor sales performance, compare product categories, analyze trends over time, and explore business data through dynamic visualizations.

## 🛠️ Tools & Technologies
- Power BI
- Microsoft Excel
- DAX (Data Analysis Expressions)
- Data Modeling
- Business Intelligence Reporting
  
## Dashboard Features
- Interactive KPI cards
- Sales trend analysis
- Product performance visualization
- Category and regional comparisons
- Dynamic slicers and filters
- Interactive business reporting
